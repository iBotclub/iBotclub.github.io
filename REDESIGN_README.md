## ðŸŽ¨ Customization Guide

### Colors
Edit in `css/modern-base.css` under `:root`:
```css
--primary-red: #ff0844;
--electric-blue: #00f0ff;
--cyber-purple: #bf00ff;
```

### Fonts
Currently using:
- Orbitron (Display)
- Exo 2 (Headings/Body)
- Space Mono (Code)

Change in `css/modern-base.css`:
```css
--font-display: 'Orbitron', ...;
--font-heading: 'Exo 2', ...;
```

### Animations
Control animation speeds in `:root`:
```css
--transition-fast: 0.2s;
--transition-base: 0.4s;
--transition-slow: 0.7s;
```
