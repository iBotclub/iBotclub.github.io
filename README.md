## ğŸ¨ Customization

### Change Colors:
Edit `css/modern-base.css`:
```css
:root {
  --primary-red: #ff0844;     /* Change this */
  --electric-blue: #00f0ff;   /* Change this */
  --cyber-purple: #bf00ff;    /* Change this */
}
```

### Change Fonts:
Edit `css/modern-base.css`:
```css
:root {
  --font-display: 'Orbitron', ...;
  --font-heading: 'Exo 2', ...;
}
```

### Adjust Animation Speed:
Edit `css/modern-base.css`:
```css
:root {
  --transition-base: 0.4s;  /* Make slower or faster */
}
```
